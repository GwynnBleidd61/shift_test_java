# File Content Filter Utility

Консольная утилита на Java для фильтрации содержимого текстовых файлов по типам данных
(целые числа, вещественные числа, строки) с записью результатов в отдельные файлы
и выводом статистики.

---

## Возможности

* Принимает **несколько входных файлов** через аргументы командной строки
* Читает данные **построчно**, в порядке перечисления файлов
* Разделяет данные по типам:

    * целые числа → `integers.txt`
    * вещественные числа → `floats.txt`
    * строки → `strings.txt`
* **Ленивое создание файлов**: выходной файл создаётся только при наличии данных соответствующего типа
* Поддержка режимов **append / overwrite**
* Вывод **краткой или полной статистики**
* Корректная обработка ошибок без аварийного завершения программы

---

## Требования

* **Java:** 17+
* **Система сборки:** не используется (чистый Java-проект)
* **Сторонние библиотеки:** не используются

---

## Запуск

```bash
java -jar app.jar [options] <inputFiles...>
```

### Опции командной строки

| Опция         | Описание                                                                  |
| ------------- | ------------------------------------------------------------------------- |
| `-o <path>`   | Путь для выходных файлов (по умолчанию — текущая директория)              |
| `-p <prefix>` | Префикс имён выходных файлов                                              |
| `-a`          | Режим добавления в существующие файлы (append). По умолчанию — перезапись |
| `-s`          | Краткая статистика (только количество элементов)                          |
| `-f`          | Полная статистика                                                         |

> Если указаны `-s` и `-f` одновременно, используется последний из них.

---

## Пример

### Входные файлы

**in1.txt**

```
Lorem ipsum
45
3.14
```

**in2.txt**

```
Hello
-1
1.0E-3
```

### Запуск

```bash
java -jar app.jar -f -p result- in1.txt in2.txt
```

### Результат

* `result-integers.txt`

```
45
-1
```

* `result-floats.txt`

```
3.14
1.0E-3
```

* `result-strings.txt`

```
Lorem ipsum
Hello
```

В консоль будет выведена статистика по каждому типу данных.

---

## Обработка ошибок

* Ошибки чтения отдельных файлов не прерывают работу утилиты
* Все ошибки аккумулируются и выводятся в `stderr` после завершения обработки
* При критических ошибках (например, неверные аргументы) программа выводит сообщение об ошибке и инструкцию по запуску

---

## Примечания по реализации

* Классификация чисел поддерживает **большие целые числа** и **научную нотацию** для вещественных
* Статистика считается независимо от успешности записи в файлы
* Код написан с использованием принципов **TDD** и ориентирован на читаемость и расширяемость
